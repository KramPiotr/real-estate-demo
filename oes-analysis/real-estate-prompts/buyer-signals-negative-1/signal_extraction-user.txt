# Negative Buyer Signals Extraction Task

## Instructions
1. Read the complete sales call transcript
2. Identify negative signals/barriers across all six categories
3. Score severity on a 1-5 scale
4. Provide direct quotes as evidence
5. Recommend actions for significant barriers (score 3+)

## Output Format
```json
{
  "negative_signals": [
    {
      "category": "financial_barriers|decision_paralysis|external_blockers|unrealistic_expectations|competitor_preference|disengagement_signals",
      "severity": 1-5,
      "quote": "exact quote from transcript",
      "transcript_range": "line_start-line_end",
      "explanation": "why this is a barrier",
      "recommended_action": "suggested response (for severity 3+)"
    }
  ]
}
```

The negative signals definition and sales call transcript will be appended separately.
